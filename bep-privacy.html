English

General

BEP Hazırlama is developed by Dev Software.

The application is designed to work fully offline.

All data created within the application is stored locally on the user's device. No data is transmitted to the developer or any external servers.

Data Collection

The BEP Hazırlama application does not collect, store, or share any personal data by the developer.

All processing happens locally on the user's device.

Advertising Services (Google AdMob)

The app may display advertisements provided by Google AdMob.

Google AdMob may automatically collect anonymous data such as:

• Device type and model
• Operating system information
• Ad interactions
• IP address (anonymous form)
• Ad performance data

This data is not collected, stored, or accessed by Dev Software.

This data collection is handled entirely by Google AdMob.

Data Sharing

Dev Software does not collect, store, or share any user data.

Data Security

All data remains on the user's device.

BEP Hazırlama uygulaması Dev Software tarafından geliştirilmiştir.

Uygulama tamamen çevrimdışı (offline) çalışacak şekilde tasarlanmıştır.

Uygulama içerisinde oluşturulan tüm veriler yalnızca kullanıcının cihazında saklanır. Bu veriler geliştiriciye veya herhangi bir sunucuya gönderilmez.

Toplanan Veriler

BEP Hazırlama uygulaması geliştirici tarafından herhangi bir kişisel veri toplamaz, saklamaz veya paylaşmaz.

Tüm işlemler kullanıcının cihazında yerel (local) olarak gerçekleşir.

Reklam Hizmetleri (Google AdMob)

Uygulama Google AdMob reklamlarını gösterebilir.

Google AdMob, reklam sunumu için aşağıdaki gibi anonim verileri otomatik olarak toplayabilir:

• Cihaz türü ve modeli
• İşletim sistemi bilgisi
• Reklam etkileşimleri
• IP adresi (anonim şekilde)
• Reklam performans verileri

Bu veriler Dev Software tarafından toplanmaz, erişilmez veya saklanmaz.

Bu veri toplama işlemi tamamen Google AdMob tarafından gerçekleştirilir.

Veri Paylaşımı

Dev Software hiçbir kullanıcı verisini toplamaz, saklamaz veya üçüncü taraflarla paylaşmaz.

Veri Güvenliği

Tüm veriler yalnızca kullanıcının kendi cihazında saklanır.
